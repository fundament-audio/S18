# S18 — Enclosure

Sealed cabinet for the BMS 18N862.

| | |
|---|---|
| **Type** | Sealed |
| **Net volume** | ~115–125 L |
| **Alignment** | Fc ≈ 47 Hz, Qtc ≈ 0.64 |
| **Material** | 22 mm MDF (this iteration) |
| **External (W×H×D)** | 550 × 550 × 602 mm |

**Construction:** double front baffle, 3× window brace, full damping stack, corner fillets. All panels 22 mm MDF.

## Damping

- **Panels:** bitumen pads on the side walls to kill panel ring.
- **Fill:** Visaton VS-Wool acoustic wool. Fill the cabinet evenly — **not too tight, but still present** throughout. Light, even fill tames internal standing waves without over-stuffing (which would skew the sealed alignment).
